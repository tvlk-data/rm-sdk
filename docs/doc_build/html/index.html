Welcome to RM-SDK’s documentation!¶

Raring Meerkat Python SDK

Configuration¶

RM-SDK needs RMConfig object during the construction Please check the RM_SDK.RMConfig class for better understanding

RMConfig object consists of

  1. RM_GRAPHQL_API_URL (This variable is required which points to the graphql api)
  2. MODEL_UPLOAD_URI (This variable is requries for models files upload)
  3. AUTH0_CONFIG

Installation¶

pip install git+https://git@github.com/tvlk-data/rm-sdk.git@{version_no}

You can get the versions from here:

https://github.com/tvlk-data/rm-sdk/releases

class rm_sdk.client.RMConfig(config_dict)[source]¶

This is the config object class for RM-SDK which will injected to rm-sdk object Config object should have this configurations

RM_GRAPHQL_API_URL (required)

AUTH0_CONFIG (optional, if provided will use the configuration to get token)

MODEL_UPLOAD_URI (Optional, if not provided will use default GCS bucket gs://rm-project-artifacts)
